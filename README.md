# React + Vite

Quotes Wisdom

A simple and elegant React application that displays inspirational quotes and wisdom messages using API data. This project is built to practice React concepts such as components, hooks, API fetching, loading states, and dynamic rendering.

🚀 Features
Fetches quotes from an API
Displays inspirational and motivational quotes
Clean and responsive UI
Loading and error handling
Built using React and Vite
Beginner-friendly project structure

🛠️ Tech Stack
React
Vite
JavaScript
CSS
